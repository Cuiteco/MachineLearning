The objective of this project is to use data from accelerometers on the belt, forearm, arm, and dumbbell of 6 subjects to predict the type of error commited with the Weight Lifting Exercise.

1. QUESTION

Is there relation between speed change in space with the correctly ejecution of particular exercise?

2. DATA

The data for this project come from:

Velloso, E.; Bulling, A.; Gellersen, H.; Ugulino, W.; Fuks, H. Qualitative Activity Recognition of Weight Lifting Exercises. Proceedings of 4th International Conference in Cooperation with SIGCHI (Augmented Human 13) . Stuttgart, Germany: ACM SIGCHI, 2013.

For more information, visit http://groupware.les.inf.puc-rio.br/har

In particular the training data was get from : https://d396qusza40orc.cloudfront.net/predmachlearn/pml-training.csv

And validation data from (I will call it Quiz data): https://d396qusza40orc.cloudfront.net/predmachlearn/pml-testing.csv

I have reviewed the Quiz data, because this will be used to predict the values of the end quizz. I noted that there are many variables with only NAs values, so I eliminated all the variables with NAs in the first observation on the Quiz data. I checked that if there is a NA in the first observation, then all the observations have NAs.

First I check that Quiz data and training data have the same variables in the same position, except in the last one. In the last variable of Training is class, and in Quiz isproblem_id.
